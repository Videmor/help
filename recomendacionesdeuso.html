Recomendaciones de uso

Se recomienda trabajar con Google Chrome, si bien el sistema trabaja con otros navegadores, algunas funcionalidades fueron diseñadas específicamente para Chrome.

En el caso de las importaciones de archivos Excel se recomienda lo siguiente: - Importar siempre archivos grabados como Excel versión 97-2003. - Si se intenta importar un archivo previamente exportado, grábelo como 97-2003 antes de importarlo.

La resolución recomendada para las vistas de Sperant es de mínimo 1024 x 728.
